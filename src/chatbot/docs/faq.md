# Frequently Asked Questions

## What is OneTracker Technologies?

OneTracker is a shipment intelligence and tracking platform designed to provide real-time logistics visibility.

## Does OneTracker support multiple carriers?

Yes. OneTracker aggregates tracking data from multiple carriers into a unified dashboard.

## How does demo booking work?

Users can book a demo via chatbot by selecting timezone, choosing a slot, and submitting required details.

## Is booking timezone-aware?

Yes. All bookings require timezone-aware datetime values and are validated before confirmation.

## Does OneTracker provide APIs?

Yes. Developers can integrate using REST APIs for shipment tracking, analytics, and booking.
