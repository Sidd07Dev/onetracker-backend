# OneTracker Demo Booking Process

To experience the OneTracker platform, users can book a live demo through the chatbot assistant.

## Demo Booking Flow

1. Ask the chatbot for a demo.
2. Provide your timezone (e.g., Asia/Kolkata, Europe/London).
3. View available demo slots.
4. Select a valid time slot.
5. Provide:
   - Full name
   - Business email
   - Company name
   - Contact number
   - Optional message
6. Receive confirmation email.

## Important Rules

- Booking must include timezone information.
- Past bookings are not allowed.
- Only specific hourly slots are available.
- Double booking of the same slot is prevented.
- Confirmation email is sent after successful booking.

All demo times are validated and stored in a timezone-aware format.
